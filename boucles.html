Les boucles
    • for (init.;condition;incr.)
      { instructions }
    • while (condition) { instructions }

  • Éléments (significations)

    • init. : var compteur = val. de départ
    • cond. : compteur < val. d'arrivée
    • incr. : compteur++

  • Exemples

    • for (var i=0; i<10; i++) { //10 itér.
        alert(i); // Affiche 0, 1...
      }
    • var i=0;
      while (i<10) { //10 itérations
        alert(i); // Affiche 0, 1...
        i++;
      }
