Les identificateurs
  • Servent à identifier une variable, une constante, une fonction ou une classe d'objet

  • Règles

    • Noms significatifs
    • Pas d'espaces ni d'accents
    • Lettres minuscules, lettre majuscules, chiffres, caractère de soulignement ( _ )
    • Ne doit pas commencer par un chiffre
    • Ne doit pas être un mot réservé

  • Conventions

    • Pleine majuscule: constante. PI
    • Majuscule initiale: classe d'objet. Date
    • Soulignement initial: propriété privée. _x

  • Séparation des mots

    • Séparés par un soulignement (snake case). ma_variable_globale
    • Distingués par une majuscule initiale (camel case). maVariableGlobale

Opérateurs

  • Règles générales

    Deux expressions ne peuvent jamais être directement juxtaposées. Un opérateur doit s'interpose entre ces deux expressions pour en donner une troisième. Une opération peut donc également être évaluée.
    Syntaxe : expression opérateur expression

    Opérateurs mathématiques

    • + addition, - soustraction
    • * multiplication, / division, % modulo

  • Opérateurs de comparaison

    • < plus petit, <= plus petit ou égal
    • > plus grand, >= plus grand ou égal
    • == égal, === strictement égal
    • != différent, !== strictement différent

  • Opérateur de chaine

    • + concaténation

  • Opérateurs booléens et de bits

    • && ET logique, || OU logique
    • & ET binaire, | OU binaire, ^ OU exclusif binaire
    • << décalage gauche, >> décalage droite

  • Opérateurs d'attribution

    Syntaxe: variable opérateur expression
    • = Attribution. Met la valeur de l'expression dans la variable.
    • += Accumulation. Ajoute à la valeur actuelle de la variable celle de l'expression.
      Les opérateurs -=, *=, /=, %=, &=, |=, ~=, <<=, >>= ont un fonctionnement similaire.
    • ++variable, variable++ pré et post incrémentation
    • --variable, variable-- pré et post décrémentation

  • Opérateurs unitaires

    Syntaxe: opérateur expression
    • + positif
    • - négatif. Inverse le signe
    • ! négation. Inverse la valeur booléenne.
    • ~ inversion de bits

  • Opérateur ternaire

    Syntaxe:
    (condition) ? valeur si true : valeur si false

Définitions
  • déclaration : Annonce de l'utilisation éventuelle d'une variable. Utilise le mot-clé var.
  • initialisation : Attribution de la valeur initiale d'une variable. Est habituellement jumelée à la déclaration.
  • expression : Portion de code pouvant être évaluée (avoir une valeur). On peut donc également déterminer son type de donnée.
  • instruction : Étape individuelle d'exécution du programme. Se termine par un point-virgule. Pour être utile, une instruction doit modifier une variable ou occasionner un affichage.
  • condition : Expression pouvant être évaluée à true ou false.
  • identificateur Mot unique désignant une variable, constante ou fonction.
Vocabulaire
  • itération : Chaque exécution d'une boucle.
  • incrémenter : Augmenter de 1 la valeur d'une variable. Peut également désigner toute augmentation d'une variable, et même la décrémentation de celle-ci.
  • compteur : Variable qui tient le compte des itérations d'boucle.
  • accumulateur : Variable destinée à
  • concaténation : Joindre diverses expressions pour former une chaine.
